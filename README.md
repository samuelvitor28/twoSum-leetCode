# Resolução do Exercício TwoSum do LeetCode usando Java

## O método TwoSum receberá um array de números inteiros "nums" e um número inteiro "target", e irá retornar os índices de dois números do array cuja soma seja igual ao valor de target.

## Regras:

  1. Cada entrada possui exatamente uma solução.

  2. Não é permitido usar o mesmo elemento duas vezes.

  3. A resposta pode ser retornada em qualquer ordem.
